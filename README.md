# job4j_car_accident [![Build Status](https://travis-ci.com/eRqa/job4j_car_accident.svg?branch=main)](https://travis-ci.com/eRqa/job4j_car_accident)

    - Hibernate;
    - Maven;
    - Tomcat;
    - Git.
